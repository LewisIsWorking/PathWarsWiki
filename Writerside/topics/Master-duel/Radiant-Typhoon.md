# Radiant-Typhoon

> Swen.
>> 1. Swen special summon itself -> Add chant.
>> 2. Chant -> Add Eldam.
>> 3. Eldamn -> Special summon itself.
>> 4. Eldam -> Adds Krosea.
>> 5. Krosea -> Tribute summon using either Swen or Eldam.
>> 6. Krosea -> Add MST & Ascendance.
>> 7. Link -> Varuroon using Krosea & Eldam/Swen.
>> 8. Varuroon -> Add MST.
>> 9. Ascendance 2nd effect -> CHAIN MST to destroy Ascendance.
>> 10. This activates Varuroon & Ascendance's in-grave effect.
>> 11. Ascendance -> Sets itself.
>> 12. Varurooon -> Sets mandate.
>> 13. Set 2 MST.
>> 14. Pass.
>> 15. On opponent's turn...
>> 16. Ascendance -> Chain MST to destroy Ascendance and summon Krosea.
>> 17. Krosea -> Add Vibrant Vortex.
>> 18. Activate Vibrant Vortex when you need a monster negate.

> Eldam.
>> 1. Eldam -> Special summon itself.
>> 2. Eldam -> Add Krosea.
>> 3. Krosea -> Tribute summon using Eldam.
>> 4. Krosea -> Add MST & Swen.
>> 5. Swen -> Special summon itself.
>> 6. Swen -> Add Ascendance.
>> 7. Link -> Varuroon using Krosea & Eldam/Swen.
>> 8. Varuroon -> Add MST.
>> 9. Ascendance 2nd effect -> CHAIN MST to destroy Ascendance.
>> 10. This activates Varuroon & Ascendance's in-grave effect.
>> 11. Ascendance -> Sets itself.
>> 12. Varurooon -> Sets mandate.
>> 13. Set 2 MST.
>> 14. Pass.

> Eldam & Krosea.
>> 1. Eldam -> Special summon itself.
>> 2. Eldam -> Add Swen.
>> 3. Swen -> Special summon itself.
>> 4. Swen -> Add Chant.
>> 5. Chant -> Add Meghala.
>> 6. This triggers Krosea to summon itself (no tribute).
>> 7. Krosea -> Add MST & Ascendance.
>> 8. Meghala -> Sumon itself.
>> 9. Meghala -> Add Ascendance.
>> 10. Activate Ascendance to add MST -> CHAIN MST to destroy Ascendance.
>> 11. Meghala -> Summons vibrant vortex.
>> 12. XYZ -> Totem bird using any level 3 monsters.
>> 13. Link -> Varuroon using a level 3 & a level 6.
>> 14. Pass. On opponent's turn...
>> 15. Activate Ascendance to summon Krosea.
>> 16. Chain link 1 Krosea, chain link 2 Varuroon.
>> 17. Varuroon -> Sets mandate.
>> 18. Krosea -> Adds Fonix.

> Krosea & Vision.
>> 1. Vision -> Add MST.
>> 2. Triggers Krosea to summon itself (no tribute).
>> 3. Krosea -> Add Chant & MST.
>> 4. Chant -> Add Eldam.
>> 5. Eldam -> Special summon itself.
>> 6. Eldam -> Add Swen.
>> 7. Swen -> Special summon itself.
>> 8. Swen -> Add Ascendance.
>> 9. Link -> Varuroon using Krosea & Eldam/Swen.
>> 10. Varuroon -> Add MST.
>> 11. Ascendance to summon a level 3 from grave -> CHAIN MST to destroy Ascendance.
>> 12. Varuroon -> Sets mandate.
>> 13. Ascendance -> Sets itself.
>> 14. XyZ -> Totem bird using any level 3 monsters.
>> 15. Mandate shuffle 3 to draw 1.
>> 16. Set MST.
