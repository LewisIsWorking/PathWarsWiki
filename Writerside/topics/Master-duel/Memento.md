>> - 18: Creation King effect -> Send creation king, combined creation and Akihiron from deck to grave.
>> - 19: Activate creation king in grave -> Banishes itself to add the field spell.
>> - 20: Field spell activate.
>> - 21: Activate memento fusion in grave -> Destroy witch -> Add bone party.
>> - 22: This triggers the field spell and akihiron's effect. 
>> - 23: Field spell effect -> Summons goblin.
>> - 24: Akihiron effect -> Summons itself.
>> - 25: Goblin destroys akihiron to send cranium burst AND any monster from deck to grave.
>> - 26: Akihiron effect -> Adds memento fusion to hand from banishment.
>> - 27: Bone party -> Destroy sleepy -> Summon dark blade or any other monster from deck.
>> - 28: Sleepy -> Send any monster such as hose.
>> - 29: Link Goblin & Dark blade into I:P Masquerena.
>> - 30: Combined Creation: From grave (to where self is pointing) by returning mace, sleepy, goblin, dark blade and horse.
>> - 31: Set memento fusion.
>> - 32: End your turn.
>> - 33: End phase: Field spell sets cranium burst from grave.
>> - 34: Flip cranium burst.
>> - 35: Elf revive Ghattic from grave.
>> - 36: When your opponent activates an effect combined creation revives witch.
>> - 37: Ghattic effect -> Add any monster from grave for guardian chimera.
>> - 38: Witch adds sleepy.
>> - 39: Creation king -> Destroys itself, pop 1 of your opponent's cards.
>> - 40: Sleepy summons itself from hand -> Fusion summon into twin headed dragon using itself and witch.
>> - 41: Twin headed dragon effect -> Destroy itself to add mace and goblin.
>> - 42: Twin dragon effect in grave + field spell OR instead of field spell use Akihiron.
>> - 43: Twin dragon effect in grave summons two bodies.
>> - 44: All main monster zones are full.#

>> - 47: Memento fusion -> Guardian chimera using goblin + stolen monster + monster that Ghattic added from grave (step 37).
>> - 48: Chimera pops 1 & draws 2.
