# Lunalight-zoodiac.md.

All of the following are from: https://www.youtube.com/watch?v=4z8jrj79ID4.

> Gold leo. Requires cross sheep.
>> 1. Normal summon gold leo.
>> 2. Gold leo effect -> Add silver hound to hand.
>> 3. Discard silver hound.
>> 4. Silver hound effect -> Summon kaleido chick.
>> 5. Kaleido chick effect -> Send black sheep to grave.
>> 6. Gold leo effect -> Add black sheep to hand.
>> 7. Discard black sheep to add polymerisation.
>> 8. Polymerization -> Fusion summon perfume dancer using kaleido chick and gold leo.
>> 9. Perfume dancer (chain link 1) + kaleido chick (chain link 2).
>> 10. Perfume dancer adds luna light perfume to hand, kaleido chick adds polymerisation to hand.
>> 11. Perfume -> Resurrect kaleido chick.
>> 12. Kaleido chick effect -> Send yellow martin to grave.
>> 13. Perfume in grave -> Discard anything to add Lunalight Tiger to hand.
>> 14. Lunalight Tiger in scale resurrects Yellow martin.
>> 15. Perfume dancer returns Lunalight Tiger to hand.
>> 16. XYZ Broad bull using kaleido chick and yellow martin.
>> 17. Broad bull detaches kaleido chick to add WOLF.
>> 18. Link Cross sheep using perfume dancer and broad bull.
>> 19. Lunalight Tiger resurrects Lunalight perfume dancer TO ONE OF CROSS SHEEPS POINTED ZONES.
>> 20. Cross sheep effect -> Summon kaleido chick from grave.
>> 21. Kaleido chick effect -> Send Gold leo to grave.
>> 22. Yellow martin in grave returns Tiger to hand to summon itself.
>> 23. XYZ Number 60: Dugares the Timeless using kaleido chick and yellow martin.
>> 24. Dugares effect -> detach kaleido chicken AND yello Martin to revive kaleido Chick.
>> 25. Link Little Knight using Cross Sheet and Number 60.
>> 26. Tiger resurrects black sheep.
>> 27. Kaleido chick effect -> Send Leo dancer to grave.
>> 28. Polymerisation Liger Dancer using perfume dancer, black sheep, wolf and kaleido chick.
>> 29. Black sheep adds wolf to hand.
>> 30. Wolf effect -> Banish leo dancer and everything EXCEPT silver hound from the grave to summon 2nd liger dancer.

> Zoodiac barrage. Requires cross sheep.
>> 1. Barrage destroys itself to summon Zoodiac Ratpier.
>> 2. XYZ Broad bull using Ratpier.
>> 3. Broad bull detaches Ratpier to add Tri-Brigade Fraktall to hand.
>> 4. Fraktall discards itself to send silver hound to grave.
>> 5. Silver hound summons Gold leo.
>> 6. Gold leo effect -> Add kaleido chick to hand, discard anything else.
>> 7. Kaleido chick effect -> Send black sheep to grave.
>> 8. Gold leo effect -> Add black sheep to hand.
>> 9. Discard black sheep to add polymerisation.
>> 10. Polymerization -> Fusion summon perfume dancer using kaleido chick and gold leo.
>> 11. Perfume dancer (chain link 1) + kaleido chick (chain link 2).
>> 12. Perfume dancer adds luna light perfume to hand, kaleido chick adds polymerisation to hand.
>> 13. Perfume -> Resurrect kaleido chick.
>> 14. Kaleido chick effect -> Send yellow martin to grave.
>> 15. Perfume in grave -> Discard anything to add Lunalight Tiger to hand.
>> 16. Lunalight Tiger in scale resurrects Yellow martin.
>> 17. XYZ Number 60: Dugares the Timeless using kaleido chick and yellow martin.
>> 18. Dugares effect -> Detach both -> Draw 2 cards.
>> 19. ^ This step is meant to be done with elder entity, but it takes up another extra deck slot.
>> 20. Yellow martin adds Lunalight Masquerade to hand.
>> 21. Tight resurrect Kaleido chick.
>> 22. Kaleido chick effect -> Send Wolf to grave.
>> 23. Yellow martin in grave returns Tiger to hand to summon itself.
>> 24. XYZ Tiger King using kaleido chick and yellow martin.
>> 25. Tiger King effect -> Detach kaleido chick to add Tenki to hand.
>> 26. Link Cross sheep using Number 60 and Tiger King.
>> 27. Tiger resurrect sheep to one of Cross sheep's pointed zones.
>> 28. Cross sheep effect -> Summon kaleido chick from grave.
>> 29. Tenki adds Black sheep to hand.
>> 30. Discard Black sheep to add wolf.
>> 31. Kaleido chick effect -> Send Leo dancer to grave.
>> 32. Activate masquerade but don't use it yet.
>> 33. Wolf effect -> Banish Leo dancer and everything EXCEPT silver hound from grave to summon 1st Liger Dancer.
>> 34. Masquerade effect -> Adds polymerisation to hand, then discard ANYTHING else.
>> 35. Link little knight using broad bull and cross sheep.
>> 36. Masquerade effect -> Sends emerald bird.
>> 37. Emerald bird effect -> Resurrect yellow martin.
>> 38. Polymerisation -> Fusion summon 2nd Liger Dancer using kaleido chick, perfume dancer, yellow martin and emerald bird.
>> 39. Black sheep resurrects Kaleido chick.
>> 40. XYZ Evil Swarm Nightmare (defence mode) using kaleido chick and yellow martin.
