## Girsu.

> Girsu.
>> 1. Normal summon Girsu.
>> 2. Girsu -> Sends harp horror.
>> 3. Girsu -> summon tokens.
>> 4. [Link] -> Galatea 1 using the token that Girsu spawned.
>> 5. Galatea 1 -> Discard 1 card from hand, get babel.
>> 6. Activate babel.
>> 7. [Link] -> Phoenix using Galatea 1 and Girsu.
>> 8. [Link] -> Mermaid using Phoenix.
>> 9. Mermaid -> Summons orcust nightmare.
>> 10. [Link] -> Closed moon using mermaid & orcust nightmare.
>> 11. [Link] -> FS Requiem using Closed moon.
>> 12. FS Requiem -> Summon Lacrima.
>> 13. Lacrima -> Send engraver.
>> 14. Engraver in GY -> Shuffle Closed Moon.
>> 15. [Link] -> FS Sequence using Lacrima & Engraver.
>> 16. Fusion -> Using Sequence's effect -> FS's Lacrima using Engraver & Lacrima.
>> 17. FS's Lacrima revives engraver.
>> 18. [XYZ] -> XYZ Caesar using FS's Lacrima & Engraver.
>> 19. [ORCUST TIME!]
>> 20. Harp Horror in grave -> Revive Skeleton.
>> 21. Nightmare -> Send Wand.
>> 22. Wand -> Summon harp horror.
>> 23. [Link] -> Galatea 2 using Harp Horror & Skeleton.
>> 24. Galatea 2 shuffles wand to get crescendo.
>> 25. Galatea 1 banishes Girsu.
>> 26. [Link] -> Enlilgirsu using sequence & Galatea 2.
>> 27. Skeleton -> Summon Galatea 2.
>> 28. [XYZ] -> Dingirsu using Galatea 2.
>> 29. Dingirsu attach skeleton.
>> 30. Enlilgirsu shuffle Nightmare -> Return Nightmare to the deck to steal the token.
>> 31. [Link] -> I:P Masquerena using Dingirsu & the token.
>> 32. Activate babel.
>> 33. TURN END.
>> 34. Galatea 1 -> Discard Nightmare -> Get Crown.
>> 35. Nightmare -> Boost Enlilgirsu's attack -> Send Wand.
>> 36. Wand -> Summon Crown.

>> 17. Link -> Galatea 2 using mermaid & Galatea 1.
>> 18. Nightmare -> Send harp horror.
>> 19. Harp horror -> Summon Girsu.
>> 20. Girsu -> Send skeleton.
>> 21. Activate Galatea 2 -> Shuffle harp horror -> Set Crescendo.
>> 22. LINK Enlilgirsu using Galatea 2 & Sequence.
>> 23. Skeleton -> Revive Galatea 2.
>> 24. XYZ Dingirsu using Galatea 2.
>> 25. Dingirsu attach skeleton.

>> 9. Galatea 1 -> Banish Knightmare -> Summons Galatea 1.
>> 10. Orcust Knightmare -> Sends skeleton.
>> 11. Galatea 2 -> Shuffle orcust knightmare -> Set crescendo.
>> 12. Link -> Longirsu using Galatea 2 & Galatea 1.
>> 13. Skeleton -> Revive Galatea 2.
>> 14. Xyz -> Dingirsu using Galatea 2.
>> 15. Link -> Enlilgirsu using Longirsu & Dingirsu.
>> 16. Enlilgirsu -> Add skeleton to hand -> Shuffle skeleton -> Take control of opponent's token.
>> 17. If you have a summon in hand, normal summon it then link into I:P Masquerena using the summon and token.
>> 18. Otherwise, normal summon anything and link into I:P Masquerena using the normal summon and token.
