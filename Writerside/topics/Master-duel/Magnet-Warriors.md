## Magnet Warriors.

> Plus Minus.
>> 1. Activate Plus Minus.
>> 2. Send Sigma Plus.
>> 3. Send Beta.
>> 4. Activate Beta in GY.
>> 5. Beta Adds Sigma Minus.
>> 6. Sigma Minus -> Special summons Plasma Magma -> Use Sigma Minus & Beta.
>> 7. ^ Put Plasma Magnum in defense mode.
>> 8. Plasma Magnum adds Magnet bonding.
>> 9. Bonding adds Epsilon.
>> 10. Plasma Magnum -> Sends Tellusion -> Normal Summons Epsilon.
>> 11. Epsilon sends Omega Plus.
>> 12. Epsilon summons Sigma Minus.
>> 13. Activate eOmega Plus in the GY to revive and add back minus.
>> 14. Use Omega PLus to make Gallant Granite so it won't get banished.
>> 15. Gallant Granite -> Adds Supplier.
>> 16. Cross Sheep using Gallant Granite and Omega Plus.
>> 17. Cross sheep must go in a zone that points to Plasma Magnum.
>> 18. Supplier -> Special summon itself to the other cross sheep zone.
>> 19. Chain link 1 Supplier, chain link 2 Cross Sheep.
>> 20. Cross sheep -> resurrect anything.
>> 21. Supplier adds Guardian.
>> 22. Tellusion -> Special summon itself.
>> 23. Banish sigma plus & sigma minus.
>> 24. Link -> I:P Masquerena using Supplier and Epsilon.
>> 25. Normal summon Guardian.
>> 26. End Board: I:P Masquerena, Guardian, Plasma Magnum, Tellusion, Cross sheep.

>> 15. Gallant Granite -> Adds Supplier.
>> 16. Tellusion -> Special summon itself by banishing Sigma plus & sigma minus.
>> 17. Link -> Cross Sheep using Gallant Granite and Omega Plus.
>> 18. Chain link 1 Supplier, chain link 2 Cross Sheep.
>> 19. Cross sheep -> resurrect anything.
>> 20. Supplier adds Guardian.
>> 21. Link -> I:P Masquerena using Supplier and Epsilon.
>> 22. Normal summon was already used so no Guardian.
