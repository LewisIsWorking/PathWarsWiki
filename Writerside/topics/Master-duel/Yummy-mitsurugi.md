# Yummy-mitsurugi

https://youtu.be/m0esG0-GFhY?si=QNoXYUTNoegZf-5B

> Any yummy card + 1 card for discard.
>> - Normal summon 1 yummy card.
>> - Link into Yummy Snatchy.
>> - Snatch places Mignon.
>> - Mignon -> resurrects the yummy card from the grave.
>> - Synchro summon into Cupsy Yummy Way using the yummy link and the resurrected yummy card.
>> - Yummy Way effect -> ...
>> - If your normal summon was Cupsy, add Cookie and Marshmallow, discard any other card to be discarded.
>> - If your normal summon was Cookie, add Cupsy and Marshmallow, discard any other card to be discarded.
>> - If your normal summon was Marshmallow, add Cupsy and Cookie, discard any other card to be discarded.
>> - Cupsy/Cookie/Marshmallow effect -> Special summon itself from hand.
>> - SYNCHRO Shamisen Samsara Sorrow Cat in DEFENCE position using Cupsy Yummy Way and the other yummy card.
>> - SYNCHRO summon Herald of arc light -> Shamisen Samsara Sorrow Cat + the other yummy card.
>> - LINK summon Gravity controller using Herald of arc light.
>> - Herald of arc light effect -> Add Ame No Habakiri No Mitsurugi. 
>> - Habakiri effect -> Special summon Mitsurugi Saji.
>> - Saji effect -> Add Mitsurugi Ritual.
>> - Mitsurugi Ritual -> Special summon Murakumo from deck TRIBUTING Mitsurugi Habakiri.
>> - Habakiri effect -> Add back Mitsurugi Mirror to hand from deck.
>> - Yummy surprise -> Resurrect marshmallow from grave.
>> - Marshmallow adds back yummy surprise to hand.
>> - Synchro summon Lollipop Yummy Way using Yummy Snatchy and Marshmallow.
>> - Lollipop Yummy Way effect -> Resurrect any two yummy cards from grave.
>> - Samsara Sorrow Cat effect -> Add Herald of arc light from grave to extra deck -> Which then Special Summons Samsara Sorrow Cat from grave.
>> - SYNCHRO summmon Herald of Arc light using Samsara Sorrow Cat and the other yummy card.
>> - Mitsurugi Mirror -> Resurrect Mitsurugi Habakiri from grave TRIBUTING Mitsurugi Murakumo.
>> - Mitsurugi Murakumo effect -> Adds Mitsurugi Prayers & resurrects itself from grave.
>> - LINK summon I:P Masquerena using Mitsurugi Habakiri and any non-yummy-way yummy monster.

> Mitsurugi Habakiri one card combo.
>> - Habakiri effect -> Special summon Mitsurugi Saji -> Tribute Saji -> Add Mitsurugi Ritual.
>> - Mitsurugi Ritual -> Special summon Murakumo from deck TRIBUTING Mitsurugi Habakiri.
>> - Habakiri effect -> Add back Mitsurugi Mirror to hand from deck.
>> - Mitsurugi Mirror -> Resurrect Mitsurugi Habakiri from grave TRIBUTING Mitsurugi Murakumo.
>> - Mitsurugi Murakumo effect -> Adds Mitsurugi Prayers & resurrects itself from grave.
>> - XYZ Zombie Vampire using Mitsurugi Habakiri & Mitsurugi Murakumo.
>> - Special summon whatever Zombie vampire's effect adds to the grave.
>> - LINK summon Gravity controller using Zombie Vampire.
>> - SYNCHRO summon Cupsy Yummy Way using Gravity Controller and whatever Zombie Vampire's effect summoned from the grave.
>> - Cupsy Yummy Way effect -> Add Cupysy and Marshmallow, discard any other card that's not the other cupsy or prayers.
>> - Cupsy special summons itself from hand.
>> - Cupsy effect -> Add Yummy Surprise.
>> - Marshmallow special summons itself from hand.
>> - LINK summon into Yummy Snatchy using Cupsy.
>> - Yummy Snatchy effect -> Place Mignon from deck to field.
>> - Mignon effect -> Resurrects Cupsy from grave.
>> - SYNCHRO summon Shamisen Samsara Sorrow Cat using Cupsy Yummy Way and Cupsy.
>> - SYNCHRO Lollipop Yummy Way using Yummy Snatchy and Marshmallow.
>> - Lollipop Yummy Way effect -> Resurrect Cupsy & Marshmallow from grave.
>> - SYNCHRO summon Herald of Arc light -> Using Shamisen Samsara Sorrow Cat & Marshmallow.
>> - Yummy surprise -> Resurrect marshmallow from grave.
>> - Marshmallow adds back yummy surprise to hand.
>> - LINK summon I:P Masquerena using Yummy Cupsy & Yummy Marshmallow.
>> - Set Yummy Surprise & Mitsurugi Prayers.
