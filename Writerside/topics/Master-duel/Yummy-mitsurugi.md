# Yummy-mitsurugi

> Any yummy card + 1 card for discard.
>> - Normal summon 1 yummy card.
>> - Link into Yummy Snatchy.
>> - Snatch places Mignon.
>> - Mignon -> resurrects the yummy card from the grave.
>> - Synchro summon into Yummy Way using the yummy link and the resurrected yummy card.
>> - Yummy Way effect -> Add Cup and Marshmallow, discard card to be discarded.
>> - Cup effect -> Special summon itself from hand.
>> - Cup effect -> Add Yummy Surprise.
>> - Synchro Metal Marcher -> Using Yummy Way and Marshmallow.
>> - Metal Marcher effect -> Summon [Cupsy Yummy Way] from grave.
>> - Synchro Herald of Arc light (to the middle monster zone) -> Using Cup and Metal Marcher.
>> - Link summon into Yummy Snatchy using Cupsy Yummy Way.
>> - Yummy surprise -> Resurrect marshmallow from grave.
>> - Marshmallow adds back yummy surprise to hand.
>> - Synchro summon Lollipop Yummy Way using Yummy Snatchy and Marshmallow.
>> - Lollipop Yummy Way effect -> Resurrect Cupsy & Marshmallow from grave.
>> - Link summon spright elf using Cupsy and Lollipop Yummy Way.
>> - Spright elf should point to Herald of Arc Light.
>> - Spright elf effect -> Resurrect Lollipop Yummy Way from grave.
