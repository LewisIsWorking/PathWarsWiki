# Maliss-in-underground

## Radiant Typhoon.

> Swen.
>> 1. Special summon swen.
>> 2. Add chant.
>> 3. Chant -> Add Eldam.
>> 4. Eldam -> Add Swen.
>> 5. Link Swen & Eldam into cherubini.
>> 6. Cherubini -> Send door mouse to grave.
>> 7. Normal summon swen.
>> 8. Link Cherubini & Swen into Lizard dose (Or Little knight if you don't have lizard dose available).
>> 9. Lizard -> Banish door mouse -> Target Lizard dose, boosting lizard dose's attack.
>> 10. Door mouse -> Summons itself.
>> 11. Door mouse -> Banish rabbit.
>> 12. Rabbit summons itself.
>> 13. Rabbit sets TB-11.
>> 14. Link -> White Binder using lizard dose & door mouse.
>> 15. DO NOT ACTIVATE WHITE BINDER ON SUMMON.
>> 16. White Binder SET TRAP -> GWC-06.
>> 17. Activate TB-11 -> Banish White Binder.
>> 18. TB-11 summons Cat.
>> 19. White Binder summons itself, drawing 1.
>> 20. White binder banishes TB-11.
>> 21. Link -> Wiz Kid using White binder & Rabbit.
>> 22. Link -> Link decoder using Cat.
>> 23. Wiz Kid banishes cat to search backup @ignister.
>> 24. Cat summons itself.
>> 25. Backup summons itself.
>> 26. Backup adds March Hare, discard anything that's not march hare.
>> 27. Cat banishes march hare to draw 2.
>> 28. March hare adds itself to hand.
>> 29. Link -> Hearts Crypter using Cat, Backup & Link Decoder.
>> 30. Link decoder summons itself.
>> 31. Link -> Firewall dragon using Hearts Crypter  & Link Decoder.
>> 32. March Hare (in hand) banishes Heart Crypter.
>> 33. March hare summons itself.
>> 34. Heart Crypter summons itself.
>> 35. Link -> Splash mage using Crypter & March Hare.
>> 36. Splash mage -> Revive anything (door mouse in this example)
>> 37. Link -> Allied Code Talker using everything except firewall dragon.
>> 38. ^ Summon allied code talker in the EMZ ABOVE firewall dragon.
>> 39. Allied code talker revives White Binder.
>> 40. END TURN.
>> 41. Allied Code talker tributes binder.
>> 42. GWC revives white binder.
>> 43. White binder -> Banish Hearts Crypter, March Hare & Rabbit.
>> 44. This triggers all 3.
>> 45. Chain link 1 Heart Crypter, Chain Link 2 White Rabbit, Chain Link 3 March Hare.
>> 46. March hare adds itself to hand.
>> 47. Rabbit summons itself.
>> 48. Heart Crypter revives itself.
>> 49. Rabbit sets MTP.
>> 50. When you want to banish your opponents card, MTP -> Banish White Binder.
>> 51. Search March Hare, then banish their card.
>> 52. White binder summons itself, draw 1.
>> 53. You also have...
>> 54. Heart Crypter to shuffle a banish card (TB-11 in this example) to banish one of their cards.
>> 55. And
>> 56. Firewall dragon to negate a card.

> Chant.
>> 1. This combo is almost the same as the above Swen combo.
>> 2. Add Eldam.
>> 3. Eldam special summons itself.
>> 4. Add Swen.
>> 5. Swen special summon.
>> 6. If you have a quick play spell, search vision.
>> 7. If you do NOT have a quick play spell, add chant to add MST.
>> 8. Activate vision.
>> 9. Draw 2, discard quick play (chant).
>> 10. Link into Cherubini.
>> 11. Cherubini sends door mouse.
>> 12. Normal summon any hand trap, summon lizard dose.
>> 13. Then it's the same as the swen combo.

> Eldam.
>> 1. Summons itself.
>> 2. Add Swen.
>> 3. Swen adds chant.
>> 4. Chant adds swen.
>> 5. Make Cherubini using eldam & swen.
>> 6. Cherubini sends door mouse.
>> 7. Normal summon swen.
>> 8. Make lizard dose using Cherubini & swen.

>> 2. Banish rabbit.
>> 3. Rabbit summon itself.
>> 4. Set TB-11.
>> 5. TB-11 banish door mouse, summon cat.
>> 6. Door mouse comes back.

>> 10. Cat summons itself.

>> 12. Add march hare.
>> 13. Discard anything self.
>> 14. Cat banishes march hare.
>> 15. March hare adds itself to hand.
>> 16. Link -> Crypter using decoder, cat & backup.
>> 17. Decoder revives itself.
>> 18. Splash mage using decoder & crypter.
>> 19. Splash mage -> Revive Maliss non-extra deck monster.
>> 20. Link -> white binder using splash mage & the revived maliss monster.
>> 21. White binder -> Banish Crypter & TB-11.
>> 22. Crypter revives itself (underneath wicked)
>> 23. White binder sets GWC-06.
>> 24. Link into transcode talker into the EMZ using wicked & binder.
>> 25. Transcode revives mage.
>> 26. Summon white hare by banishing white binder.
>> 27. White binder comes back and draws 1.
>> 28. Link -> Firewall dragon using binder & mage below Transcode talker.
>> 29. Link -> Allied code talker using Crypter, Transcode & March Hare (everything EXCEPT firewall).
>> 30. ^ Summon it to the EMZ (above firewall).

>> 32. ^ Transcode on the left, binder on the right.
>> 33. END TURN.

> Maliss P Dormouse. 
>> - 1 = Hand -> Normal-summon Dormouse -> Dormouse (field) banishes March hare (deck).
>> - 2 = Activate march hare (banishment) effect -> Add march hare from banishment to hand.
>> - 3 = Link dormouse into link decoder.
>> - 4 = Activate march hare effect -> Banish dormouse.
>> - 5 = The Dormouse effect -> Summon dormouse from banishment to field.
>> - 6 = Link decoder, dormouse and march hare into red ransom.
>> - 7 = Red Ransom effect + chain decoder's effect.
>> - 8 = Red Random's effect adds Maliss in Underground to hand.
>> - 9 = Red Random + Link decoder into Apollousa.
>> - 10 = Active Maliss in Underground effect -> Banish mirror.
>> - 11 = Mirror banishes red ransom.
>> - 12 = Mirror Adds Rabbit to hand.
>> - 13 = Red Ransom summons itself and banishes the cat.
>> - 14 = Cat summons itself.
>> - 15 = Cat banishes rabbit.
>> - 16 = Draw 2.
>> - 17 = Rabbit summons itself.
>> - 18 = Rabbit effect sets GWC6 from deck.
>> - 19 = Link Red Random and either Rabbit or Cat into splash mage.
>> - 20 = Splash mage effect -> revive any Maliss monster.
>> - 21 = Link Splash mage and revived Maliss into White binder.
>> - 22 = DO NOT ACTIVATE WHITE BINDER'S EFFECT.
>> - 23 = Active GWC6 -> Banish White Binder -> Summon Red Ransom.
>> - 24 = Active the white binder effect in banishment -> Summons itself.
>> - 25 = DO NOT ACTIVATE WHITE BINDER'S EFFECT.
>> - 26 = The White binder -> Set GWC6 from grave to field.
>> - 27 = Link everything into Terror hertz (OR DeCrypt if you do not play terror hertz).
>> - 28 = During the end phase terror hertz effect -> Send Wyrm to graveyard.
>> - 29 = On the opponent's turn the trap lets you summon the white binder to banish the opponent's graveyard.
>> - 30 = Use white binder to also banish red random and sometimes rabbit.
>> - 31 = Rabbit and Red Ransom can summon themselves back.
>> - 32 = Red Ransom banishes Dormouse or another card.
>> - 33 = Red Ransom, Rabbit and Dormouse all activate.
>> - 34 = Red Ransom adds Maliss in Underground to hand.
>> - 35 = Rabbit sets MTP-07 from deck to field.
>> - 36 = Active MTP-07 -> Banish White Binder -> to banish an opponent's on field card.
>> - 37 = White binder comes back to draw 1 card.
>> - 38 = Terror hertz effect -> Send pack bit to graveyard -> Pack bit's effect pushes a monster to the back row.

>> - 1 = Banish dormouse.
>> - 2 = Summon dormouse.
>> - 3 = Banish rabbit.

>> - 5 = Set GWC6.

>> - 9 = Active the red ransom -> Add Mirror.
>> - 10 = Active GWC6 -> Banish red ransom -> Summon rabbit from grave DO NOT SELECT RED RANSOM.

>> - 12 = Cat summons itself.
>> - 13 = Cat banishes mirror -> Draw 2.
>> - 14 = Mirror banishes dormouse to-search the March Hare.
>> - 15 = Normal-summon the March Hare.
>> - 16 = Link March hare into decoder.
>> - 17 = Link decoder, rabbit and cat into white-binder.
>> - 18 = Activate white binder, chain decoder's effect -> With white binder's effect target march hare.
>> - 19 = Decoder summons itself.
>> - 20 = March hare summons itself.
>> - 21 = White binder sets GWC6 from deck.
>> - 22 = Link decoder and white binder into Firewall dragon. Put firewall dragon into column 4.
>> - 23 = Summon March Hare on the left from red ransom. Put March Hare in column 2.
>> - 24 = March hare is banishing white binder.
>> - 25 = White binder summons itself. Column 5.
>> - 26 = Draw 1.
>> - 27 = March hare links into Linguriboh.
>> - 28 = Firewall adds March hare and cat to hand.
>> - 29 = Linguriboh + Red ransom + White binder into Maliss<Q>Hearts Crypter. Crypter into the extra monster zone above fireball dragon.
>> - 30 = This triggers Firewall dragon to summon Cat from the hand. Column 3.
>> - 31 = Crypter + Firewall + Cat into Terror Hertz.
>> - 32 = Set cards.
>> - 33 = End turn.
>> - 34 = End phase: Terror hertz effect -> Send Wyrm to graveyard.
>> - 35 = During your opponent's turn when they have something to banish from the graveyard-> Activate GWC6.

>> - 38 = Red ransom summons itself.
>> - 39 = Crypter summons itself.
>> - 40 = Red ransom banishes rabbit in the grave.
>> - 41 = Rabbit comes back to set MTP-07 from deck.
