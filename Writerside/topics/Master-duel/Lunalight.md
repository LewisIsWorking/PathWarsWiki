# Luna Light

## Combo 1.

Gold Leo -> Add Yellow Martin -> Discard Yellow Martin.

Yellow Martin -> Add Masquerade -> Discard Silver Hound.

Silver hound -> Summon Kaleido Chick.

Kaleido Chick's effect -> Send black sheep to grave.

Gold leo adds black sheep to hand.

Discard Black sheep to add polymerization.

Polymerization -> Fusion summon perfume dancer using kaleido chick and gold leo.

Masquerade (chain link 1) + Perfume dancer (chain link 2) -> Discard perfume.

Perfume dancer effect -> Return masquerade to hand.

Perfume in grave -> Discard masquerade to add Tiger (pendulum monster).

Tiger column 1 -> Resurrect Kaleido chick.

Kaleido chick effect -> Send Leo Dancer to grave.

Link summon S:P Little Knight using perfume dancer and kaleido chick.

Yellow martin in grave -> Return tiger to hand.

Tiger again -> Resurrect Kaleido chick.

Kaleido chick effect -> Send Gold Leo (or black sheep) to grave.

Kaleido chick + Yellow martin -> Xyz into Granite.

Polymerization Liger dancer using both gold leos, the leo dancer and 1 black sheep in the grave.

End board: S:P Little Knight, Granite, Liger Dancer.
