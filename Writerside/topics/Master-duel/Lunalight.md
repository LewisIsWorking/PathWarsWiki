## Combo 1. Gold Leo first.

> Normal summon Gold Leo ->
>> 1. Add & Discard silver hound (do not activate Gold Leo).
>> 2. Silver hound effect -> Summon kaleido chick.
>> 3. kaleido chick effect -> Send black sheep to grave.
>> 4. Gold leo effect -> Add black sheep to hand.
>> 5. Discard black sheep to add polymerisation.
>> 6. Polymerization -> Fusion summon perfume dancer using kaleido chick and gold leo.
>> 7. Perfume dancer (chain link 1) + kaleido chick (chain link 2).
>> 8. Perfume -> Resurrect kaleido chick.
>> 9. kaleido chick effect -> Send leo dancer to grave.
>> 10. Perfume dancer effect -> Return kaleido chick to hand then resummon kaleido chick.
>> 11. kaleido chick effect -> Send Yellow martin to grave.
>> 12. Perfume in grave -> Discard polymerisation to add Luna light wolf to hand.
>> 13. Perfume dancer + kaleido chick -> Link summon Cross sheep.
>> 14. Wolf into pendulum zone.
>> 15. Wolf's effect -> Fusion summon Liger Dancer (to cross sheep's pointed zones) using both fusions (perfume dancer & leo dancer), black sheep and gold leo.
>> 16. Cross sheep effect -> summon kaleido chick from grave.
>> 17. kaleido chick effect -> Send 2nd leo dancer to grave.
>> 18. Yellow martin in grave -> Return luna light wolf to hand.
>> 19. XYZ summon Tiger King using kaleido chick and yellow martin.
>> 20. Tiger King effect -> set Tenki.
>> 21. Tenki -> Search luna light tiger.
>> 22. Tiger King effect -> detach kaleido chick.
>> 23. Luna light Tiger effect -> Resurrect kaleido chick.
>> 24. kaleido chick effect -> Send gold leo (or black sheep) to grave.
>> 25. Link summon I:P Masquerade using kaleido chick and Tiger King.
>> 26. Wolf -> Banish leo dancer + everything EXCEPT silver hound from grave.
>> 27. Final board: I:P Masquerade, TWO Liger Dancers and cross sheep.
>> 28. On your opponents turn link summon S:P Little Knight using masquerade and cross sheep.
