## Dual class macros

> How do I set up my dual class on Foundry?
>> - Go to the macro directory and search for the "Based Dual Class" macro.
>> - Execute the macro, select your two classes, then drag the dual-class item from the items directory onto your character sheet.
>> - Full step-by-step guide with pictures: [Setting-up-dual-class-FoundryVTT.md](Setting-up-dual-class-FoundryVTT.md)

## Unpause macros

> The game is paused, what do I do?
>> - You do not need to wait for the Game Master!
>> - Players can unpause the game themselves.
>> - Go to the macro directory and run the unpause macro.
>> - Tip: drag the unpause macro onto your hotbar so it is only one click away during sessions.

## Hero points and hero actions

> How do hero points work here?
>> - We use the Hero Point Deck variant rule. It is required.
>> - Drawing and trading hero point cards is done via the Foundry character sheet interface.
>> - See: [Official-paizo-variant-rules.md](Official-paizo-variant-rules.md)

## Reroll rules

> How do rerolls work?
>> - Spend 1 hero point to reroll a check. You must take the new result.
>> - House variant (via the PF2e Workbench module): if the reroll lands under 10, it gets +10.

## Point of truth

> My PathBuilder build and my Foundry sheet do not match. Which one is correct?
>> - Your Foundry character sheet is the single point of truth for your character.
>> - PathBuilder and paper sheets are planning tools only.
>> - If they ever disagree, the Foundry sheet wins.
